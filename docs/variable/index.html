    public ItemHeader(item: Resource, options: Object, isOpen: Boolean, isActive: Boolean): * source 

    import {ItemHeader} from 'nested-resource-editor/src/panels/ItemHeader.js'

    The ItemHeader component configures accordion header for a given resource. The header consists of a left side icon that indicates whether the accordion tab is open or closed, the title formed from the resource's id and name, and the right hand icon that shows the class of the resource. The is an option to include to the header a checkbox with an icon to mark the resource as active.

    Emit:

    *

    activeItemChanged - the item has been marked as active
